UX Research Case Study

The Science of the Underworld

Modeling information processing and expert decision strategy in Hades.

The Objective

This study investigates how players manage cognitive load and attentional zones during fast-paced combat, where survival and damage optimization depend on rapid filtering, prioritization, and action selection. The goal was to identify where interface signals help or hinder tactical accuracy under time pressure.

Methodology

  • Literature Review
  • Heuristic Evaluation
  • Expert Strategy Analysis

The Framework

The evaluation applies the Human Data Processor model to map how player cognition shifts during boon selection. In high-threat encounters, working memory handles immediate constraints such as health, chamber hazards, and enemy pressure. Long-term memory stores learned boon synergies, mirror upgrades, and route heuristics. Expert performance emerges when players quickly offload known patterns to long-term memory and reserve working memory for tactical adaptation.

Expert vs. Novice UX

A Fresh File speedrun baseline was used to compare cognitive strategy under limited build certainty. Experts consistently favor Divine Dash because it provides immediate defensive reliability, collision-safe repositioning, and higher damage uptime through consistent spacing control.

Key Insights

  • Choice architecture is strongest when immediate affordance value is legible at a glance.
  • Fresh File constraints expose how uncertainty increases working-memory load during boon prioritization.
  • Experts exploit stable high-value options to preserve attention for combat execution.
  • Attentional zone management predicts consistency more reliably than raw reflex speed.
